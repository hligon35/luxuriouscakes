# Wedding Cakes

Add your wedding cake photos to this folder.

Recommended naming:
- elegant-wedding-cake.jpg
- rustic-wedding-cake.jpg
- floral-wedding-cake.jpg
- multi-tier-wedding.jpg

Make sure images are:
- High quality (1200x800px minimum)
- Well-lit and clear
- Under 2MB file size
- JPG or PNG format
